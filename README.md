# Entorno virtual para Django

Este proyecto contiene un entorno virtual para trabajar con Python y Django.

Generated by Copilot

Pasos rápidos (Windows - PowerShell):

1. Activar el virtualenv en PowerShell:

   & .\.venv\Scripts\Activate.ps1

   O (si tienes ejecución de scripts restringida) ejecuta:

   PowerShell -ExecutionPolicy ByPass -File .\.venv\Scripts\Activate.ps1

2. Alternativa (CMD):

   .venv\Scripts\activate.bat

3. Comprobar que estás usando el Python del venv:

   where python

   o

   .\.venv\Scripts\python.exe -V

4. Crear un proyecto Django (ejemplo):

   .\.venv\Scripts\python.exe -m django startproject mi_proyecto .

5. Ejecutar el servidor de desarrollo:

   .\.venv\Scripts\python.exe manage.py runserver

6. Si quieres regenerar `requirements.txt` después de instalar paquetes:

   .\.venv\Scripts\python.exe -m pip freeze > requirements.txt

Notas:
- El virtualenv está en `.venv` dentro del proyecto.
- `requirements.txt` fue generado automáticamente tras la instalación inicial de Django.

Si necesitas que cree un proyecto Django base o pruebas automatizadas, dime y lo creo.
